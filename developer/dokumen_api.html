Dokumentasi API

Dokumentasi API untuk integrasi dengan Zenodo dan platform publikasi ilmiah lainnya. API ini menyediakan akses ke berbagai data publikasi, pengajuan penelitian, serta data pengelolaan sampah.

Akses Terbatas: Hanya anggota terdaftar

Endpoint Utama

GET /api/publications
Mendapatkan daftar publikasi ilmiah komunitas. Dapat difilter berdasarkan tahun, topik, dan penulis.
Parameter
Tipe
Deskripsi
year
integer
Tahun publikasi (contoh: 2023)
topic
string
Topik penelitian (contoh: "daur ulang")
author
string
Nama penulis
POST /api/submit
Mengirimkan publikasi baru ke platform. Memerlukan autentikasi dan izin khusus.
Parameter
Tipe
Deskripsi
title
string
Judul publikasi (wajib)
authors
array
Daftar penulis (wajib)
abstract
string
Abstrak penelitian (wajib)
keywords
array
Kata kunci penelitian
file
file
File PDF publikasi (wajib)
GET /api/waste-data
Mengakses data pengelolaan sampah terbaru. Menyediakan statistik daur ulang, pengurangan sampah, dan dampak lingkungan.
Parameter
Tipe
Deskripsi
region
string
Filter berdasarkan wilayah (contoh: "jawa-barat")
period
string
Periode waktu (monthly, quarterly, yearly)
